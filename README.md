# PiHole_Blocklists
This repository provides automatically maintained, categorized domain and subdomain blocklists built from multiple public sources.

The lists are continuously updated, deduplicated, and cleaned to include only valid domains, making them suitable for DNS filtering, Pi-hole, firewalls, SIEM, SOC operations, and threat intelligence pipelines.

Each category is maintained as an independent list.
